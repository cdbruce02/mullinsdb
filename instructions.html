STUDENT INSTRUCTIONS


Note: It is REQUIRED to inform Mr. Mullins of all items you take.

This will be used for keeping track of item quantities.


Map Search

To use the map feature, click on a section of the map. When clicked, you will be brought to another page displaying all of the items in that colored zone of the room along with their corresponding Item IDs, Quantities, and Locations. This feature may be used for preview of a section


Search Bar

To search for an item, please type the item you are searching for in the search bar. If nothing appears on first search, then try adding/removing an 's' or any other plural.
This will display a table with the names of all items that match the search and their corresponding Item IDs, Quantities, and Locations. Once you've found the item you're searching for, remember it's Location. Use the Display Map tab to open the map in another tab for location reference if needed. Finally, go find the Item! But remember to let Mr. Mullins know before you borrow/use the item for inventory tracking purposes.


Display Map

Clicking on the map tab in the navigation bar will bring up an image of the map in a separate browswer window. This can be used for reference when searching for an item.

TEACHER INSTRUCTIONS


Note: As students approach, use the "Edit Item" feature on the login screen to update quantities.

Login Page

To access teacher-exclusive inventory management features (add, remove, edit), click on the "Teacher Login" tab in the navigation bar. This will bring you to the login page, where you can input an assigned username and password. (just use user: testuser, pass: testpass for now) Once entered, this will create a session that will keep you logged in for the duration of the open tab. You are now free to access the login home page, which contains buttons for each feature.


Add an Item

To add an item, login using given credentials and navigate to teacher home page. Then, click on 'Add Item' button. Fill out ALL fields, including Item ID, Name, Quantity, Zone, Color, and Subsection.


Edit an Item

To edit an item, login using given credentials and navigate to teacher home page. Then, click on 'Edit item' button. Type in the Item ID of the item you would like to edit. This will bring you to a page where you can insert new values for each field. If you do not plan to change an item's attribute in one category, just copy the 'old' value and paste it in the 'new' field. Fill out ALL fields. **NOTE** For security and database continuity reasons, you are unable to edit the Item IDs from the website. For this, you must connect to the database using MySQL Workbench.


Remove an Item

To remove an item, login using given credentials and navigate to teacher home page. Then, click on 'Remove Item' button. Type in the Item ID of the item you would like to remove. Then, click submit. The item will be deleted from the inventory database.
